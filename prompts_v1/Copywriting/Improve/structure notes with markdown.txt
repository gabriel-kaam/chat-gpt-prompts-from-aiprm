- listes à puces (* liste)
- listes numérotées (1. element 1, 2. element 2, 3. element 3.....)

- code incorporé (« code incorporé ») -> Si du code est présent dans la note
- blocs de code (```code```) -> Si un script est présent dans la note
- liens ([Texte d’affichage du lien](http://www.sampleurl.com)) -> Si un lien est présent dans la note
